# 📄 AI PDF 分析器

上传 PDF → 一键摘要 → 智能问答。支持中英文 PDF。

## 功能

- 📤 上传多个 PDF，自动提取文本
- 📊 文档概览（页数、字数、文件数）
- 📝 一键 AI 摘要（核心要点 + 结论）
- 💬 智能问答（基于文档内容提问）
- ❓ 自动生成关键问题
- 🌐 多供应商支持（OpenAI / DeepSeek / Groq / Claude）

## 快速开始

```bash
pip install -r requirements.txt
streamlit run pdf-analyzer/app.py
```

## 卖点（接单时用）

> 律师、HR、研究人员每天要读大量 PDF。这个工具自动总结 + 智能问答，节省 80% 阅读时间。
